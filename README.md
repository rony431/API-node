# Node API weather + geolocation

Application with endpoints to retrieve weather and geolocation information

## Installation
Node version 12
Docker: 
Clone the project
Create .env file 
Put the keys provided in the email

```bash
npm install 
docker-compose build
docker-compose up
```
it should run on PORT:8000
```bash
localhost:8000
```

POSTMAN testing
